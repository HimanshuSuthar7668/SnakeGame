# Snake Game 🐍

A classic Snake Game developed in Java using the Swing framework as a fun college project. The player controls a snake that grows longer by eating food while avoiding collisions with the walls and its own body.

## Features

* Smooth snake movement
* Random food generation
* Score tracking
* Game Over detection
* Keyboard controls
* Built with Java Swing GUI

## Technologies Used

* Java
* Java Swing
* Object-Oriented Programming (OOP)

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/HimanshuSuthar7668/SnakeGame.git
```

2. Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, NetBeans, etc.).

3. Compile and run the main Java file.

## Controls

| Key | Action     |
| --- | ---------- |
| ↑   | Move Up    |
| ↓   | Move Down  |
| ←   | Move Left  |
| →   | Move Right |
